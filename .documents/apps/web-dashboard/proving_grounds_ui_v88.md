Física de la Interfaz: El Proving Grounds Hub actúa como un Nexo de Invocación. A diferencia de una vista de telemetría pasiva, esta página tiene autoridad para disparar procesos pesados en el backend y visualizarlos bit-a-bit.
Verbocidad Forense: Se utiliza el componente SystemLogConsole para unificar los logs. Cuando una prueba de Rust (L1) se ejecuta, envía mensajes vía handle_proving_report al orquestador, el cual los emite al Dashboard. Esto crea la ilusión de una terminal de mainframe conectada al núcleo.
UI de Alta Densidad: El diseño utiliza una paleta de negros profundos con acentos en esmeralda y azul eléctrico, optimizada para la visualización de datos técnicos de tesis doctoral.
